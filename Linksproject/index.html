This is my first HTML assignment, and I am excited to share it with you all. Learning HTML is fun and opens up a world of possibilities for creating beautiful web pages. Understanding how web pages are accessed starts with IP addresses and DNS. An IP address (IPv4) is a unique numerical identifier assigned to each device connected to the internet. It allows devices to communicate with each other over the internet. When you enter a domain name (like www.example.com) into a web browser, the Domain Name System (DNS) translates this human-readable domain name into an IP address. This translation enables your browser to locate and connect to the correct web server hosting the desired content. In HTML, you can specify links and resources using domain names, and behind the scenes, DNS ensures these names are translated to IP addresses so that browsers can fetch and display web pages correctly.
